Features

Everything you need,
nothing you don't

Built from scratch with performance and simplicity in mind. No bloat, no launcher, no account required.

⚡

Fixed-Timestep Game Loop

Physics and logic run at a fixed 8 ticks/sec while rendering interpolates for buttery-smooth 60 fps visuals.

🎮

Gamepad Support

Full controller support via the Web Gamepad API. Keyboard (arrows + WASD) and touch swipe also work out of the box.

🎨

Native Canvas2D Renderer

Custom renderer using native Canvas2D — rounded segments, animated eyes, food pulse glow. No GPU required.

🔊

Web Audio Sound Effects

Procedurally generated sound effects via Web Audio API. Mute toggle always accessible in-game.

📦

Zero Install

Just unzip and double-click Snake.exe. No installer, no admin rights, no dependencies.

🏆

High Score Tracking

In-session high score persisted across rounds. New high score celebration on the game-over screen.
